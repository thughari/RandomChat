Privacy Policy

Last Updated: June 14, 2025

Welcome to RandomChat, an innovative video chat platform developed by Hari Thatikonda. This application demonstrates the power of modern web technologies while prioritizing user privacy and seamless communication.

1. Application Overview

RandomChat connects users through:

  • Instant video chat connections
  • High-quality audio streaming
  • Secure peer-to-peer technology
  • User-friendly controls
  • Cross-platform compatibility

2. How RandomChat Works

The application operates through:

  • Automatic user pairing system
  • Direct peer-to-peer connections
  • Real-time video/audio streaming
  • Temporary session management
  • Browser-based interface (no downloads required)

3. User Privacy & Security

Your privacy is protected through:

  • End-to-end encrypted communications
  • No personal data collection
  • No registration requirements
  • No chat history storage
  • Temporary session IDs only
  • Automated session cleanup

4. Technical Architecture

Built with cutting-edge technologies:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Java 21, Spring Boot
  • Communication: WebRTC, WebSocket
  • Infrastructure: Render.com cloud hosting
  • Performance: Virtual Thread implementation

5. Usage Guidelines

To ensure the best experience:

  • Use a modern web browser (Chrome, Firefox, Safari)
  • Allow camera and microphone access when prompted
  • Maintain stable internet connection
  • Report technical issues via provided channels
  • Respect other users' privacy

6. Data Processing

During your session:

  • Video/audio streams remain peer-to-peer
  • No content is recorded or stored
  • Temporary connection data is automatically cleared
  • No cookies or tracking mechanisms used
  • No third-party services involved
